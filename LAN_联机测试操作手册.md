# 逆转 Uno 联机测试操作手册

## 1. 手册说明

本手册覆盖两种联机模式：

- 模式 A：同一局域网联机（LAN）
- 模式 B：跨网络联机（cpolar 隧道）

---

## 2. 房主端启动流程

### 2.1 启动服务端

```bash
cd ProGyakuten
npm run dev:server
```

### 2.2 启动客户端

新开一个终端：

```bash
cd ProGyakuten
npm run dev:client
```

### 2.3 本机健康检查

打开 `http://localhost:5173`，应看到标题画面。

---

## 3. 游戏流程

### 3.1 注册与登录

1. 打开页面，点击进入登录界面
2. 输入用户名和密码，点击"注册"创建新账户（或"登录"已有账户）
3. 登录成功后进入角色选择界面

### 3.2 角色选择

1. 选择已有角色，或点击"创建角色"新建（每账户最多 3 个）
2. 选择角色后进入大厅

### 3.3 创建/加入房间

1. 房主：输入房间名，点击"创建房间"
2. 其他玩家：在房间列表中点击房间加入
3. 等待玩家到齐（支持 2/4/6 人）

### 3.4 开始游戏

房主点击"开始游戏"，对局开始。

### 3.5 断线重连

任何玩家刷新页面即可自动重连，恢复到断线前的游戏状态。

---

## 4. 模式 A：局域网联机

### 4.1 房主准备

1. 执行 `ipconfig`（Windows）或 `ifconfig`（Mac），找到局域网 IPv4
2. 放行防火墙端口 5173（前端）和 3001（WebSocket）

### 4.2 其他玩家连接

- 打开 `http://房主IP:5173`
- 注册/登录后即可创建或加入房间

---

## 5. 模式 B：跨网络联机（cpolar）

### 5.1 开启隧道

需要两条隧道：
1. 前端隧道：映射 `localhost:5173`
2. WS 隧道：映射 `localhost:3001`

### 5.2 HTTPS 限制

如果通过 HTTPS 访问前端，WebSocket 必须使用 `wss://` 协议。

### 5.3 其他玩家连接

- 打开前端公网地址
- 注册/登录后即可参与

---

## 6. 常见问题

### 6.1 连接失败

- 检查服务端是否在 3001 端口运行
- 远程玩家确认 `wss://` 与页面协议匹配

### 6.2 加入房间失败

- 房间已满（最多 6 人）
- 房间已开局（需等待本局结束）

### 6.3 刷新后变成其他玩家

- 已通过 per-tab session 修复
- 确保不同账户在不同标签页中各自登录
